시스템 개요

본 시스템은 도메인 또는 IP 주소를 입력받아 자동으로 보안 스캔을 수행하고, 그 결과를 시각화하여 사용자에게 제공합니다. 사용자는 스캔 결과를 저장하거나, 타임라인 및 리포트 형태로 아카이빙할 수 있으며, 필요 시 스캔을 중단하거나 시스템을 종료할 수 있습니다.

주요 기능:

  • 정적/동적 스캔 지원
  • 자동 리포트 생성
  • 아카이빙 기능 (Timeline, Snapshot, Report)
  • 사용자 친화적인 대시보드 UI
